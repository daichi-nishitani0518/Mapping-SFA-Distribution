# FAQ

## どんなアプリですか？

1つの案件に複数の業者が絡む様な入札形式や見積もり合わせに対応したSFAアプリです。

## 何ができますか？

ダッシュボード、マインドマップ表示、案件一覧 / 詳細管理、タスク管理、カレンダー、添付ファイル、ログイン機能、横断検索、通知機能、バックアップ / 復元、AI用エクスポート、差分マージ取り込み、重複候補アラート、束ね登録に対応しています。

案件一覧テーブルではカラム幅のドラッグリサイズや、メモセルのホバーによる全文表示にも対応しています。

## タスク管理について

### タスク管理で何ができますか？

v0.3.4 からタスク管理機能が利用可能です。カンバン形式のタスクボードで、カラム（未着手 / 進行中 / 完了）をカスタマイズできます。タスクに顧客案件を紐付け、優先度・進捗率・期限を管理できます。タスク新規作成時に添付ファイルを追加することも可能です。画面上部に KPI（総数 / 完了 / 進行中 / 未着手 / 期限超過）が表示されます。

## カレンダーについて

### カレンダーで何ができますか？

v0.3.4 から左サイドメニューに「カレンダー」タブが追加されました。タスクの期限や案件の契約開始月をカレンダー形式で確認できます。祝日・営業日に対応しています。カレンダーから案件一覧への遷移・ハイライトも可能です。

## 添付ファイルについて

### タスクや案件にファイルを添付できますか？

はい。v0.3.4 からタスク・案件にファイルを添付できます。添付ファイルはローカルの IndexedDB に Blob として保存されます。タスク新規作成時にも添付可能です。PDF はアプリ内でプレビューでき、それ以外のファイルは Finder で開きます。

### 添付ファイルの容量制限はありますか？

ローカル保存のため、容量は Mac のストレージ容量に依存します。

## ログインについて

### ログインが必要ですか？

はい。v0.3.4 から初回起動時に ID / パスワードの登録が必要です。登録後は日付が変わるまでログイン状態が維持されます。

### パスワードを忘れた場合は？

設定画面からパスワードを変更できます。現在のパスワードを入力して新しいパスワードを設定してください。

## 横断検索について

### 画面上部の検索窓で何ができますか？

タスク・顧客案件・商流案件・顧客名・業者名を横断検索できます。検索結果から「案件一覧」または「マップ」を選んで遷移できます。

- 「案件一覧」を選ぶと、該当行がハイライト表示されます（詳細モーダルは開きません）
- 「マップ」を選ぶと、マインドマップで該当ノードが5秒間発光した後、選択枠に切り替わって中央に配置されます

## 通知について

### ヘッダーのベルマークは何ですか？

v0.3.4 から通知機能が追加されました。期限超過案件などの通知をベルアイコンから一覧表示できます。件数バッジで未読数が分かります。通知をクリックすると該当案件へ遷移します。

## データはどこに保存されますか？

基本はローカル環境に保存されます。外部サーバーへ案件データを送る前提ではありません。

## バックアップはできますか？

はい。JSON でバックアップを書き出しできます。必要なときはその JSON から復元できます。

## 復元はどうやりますか？

アプリ内の取り込み機能からバックアップ JSON を選択して復元します。

## AI用エクスポートとは何ですか？

案件データを、AIで整理・要約・分析しやすい形で書き出すための機能です。

## 差分マージ取り込みとは何ですか？

既存データを丸ごと消さずに、差分だけを取り込むための機能です。複数のバックアップや引き継ぎデータを統合したいときに使います。

## 重複候補アラートとは何ですか？

似た条件の案件がすでに存在する場合に候補を表示する機能です。その場で既存案件へ束ねるか、別案件として登録するかを選べます。`v0.3.2` 以降は数量 ±10% または 契約開始年月 ±3ヶ月 以内の組み合わせのみ束ね対象となります。

## 案件一覧のカラム幅を調整できますか？

はい。v0.3.3 から案件一覧の全 16 カラムをドラッグで横幅調整できます。カラム境界に縦線が表示されるので、その線をドラッグしてください。顧客名は漢字10文字分（最低 200px）、カテゴリは最低 120px が保証されます。幅は localStorage に自動保存されます。

## メモの内容を素早く確認するには？

v0.3.3 から、案件一覧のメモセルにマウスを乗せると 200ms 後に小さなポップオーバーでメモ全文が表示されます。スクロール / Esc / マウスが外れると自動で閉じます。

## ダッシュボードの新機能について

`v0.3.2` では、契約開始月を過ぎたパイプライン中の案件がある場合、KPIカード直上に赤枠の注意アラートを表示します。

## 失注に変更するときの注意点は？

`v0.3.2` では、ステータスを「失注」に変更して保存する際に確認ダイアログを表示します。チェックボックスを入れて保存することで失注に変更されます。

## アップデート時にデータはどうなりますか？

`v0.3.4` へ上書きインストールした場合、Electron 内部が HTTP サーバー方式に移行していますが、旧バージョンの IndexedDB データは自動で移行されます。既存の案件データ・タスクデータ・設定は引き継がれます。カラム幅は初回リサイズ操作までデフォルト幅で表示されます。

## どの環境で使えますか？

`v0.3.4` は macOS / Apple Silicon 向けです。Intel Mac では動作しません。

## 起動できないときは？

ad-hoc署名のみ・未公証のため、`Applications` フォルダ内の `Mapping-SFA.app` を右クリックし、`開く` を選択してください。詳しくは [INSTALL.md](./INSTALL.md) を参照してください。
