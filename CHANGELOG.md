# CHANGELOG

## 0.3.2

- ダッシュボードに期間超過案件アラートを追加
  - 契約開始月を過ぎたパイプライン案件がある場合、KPIカード直上に赤枠で注意喚起
- 案件詳細画面に今年度計上金額を上部表示
- 失注変更時の確認ダイアログを追加
  - ステータスを「失注」に変更して保存する際に「本当に失注に変更しますか？」を表示
  - チェックボックスを入れて保存することで失注に変更される
- ヘッダーのアプリ名称を画像ロゴへ変更
- タブナビゲーションをロゴ右側へ再配置
- タスク管理画面にインラインカラム追加UIを追加
- 案件詳細デザイン（バッジ / ボタン / テーブル / 情報表示）を他の画面と統一
- 重複候補アラート / 束ね登録の判定を数量±10% または 契約開始年月±3ヶ月以内に限定
- 束ねられた案件を分離する「分ける」ボタンを案件詳細に追加
- macOS / Apple Silicon 向けDMGをad-hoc署名付きで再生成

- バックアップ / 復元に対応

  - 候補案件への束ね登録
  - 別案件としての登録分岐
- 束ね登録まわりの操作性を改善
- ダッシュボード内訳表示を調整
