# CHANGELOG

## 0.3.4

### 新機能

- タスク管理機能
  - カンバン形式のタスクボード（カラム: 未着手 / 進行中 / 完了、カスタマイズ可能）
  - タスクに顧客案件を紐付け可能
  - 優先度・進捗率・期限を管理
  - タスク新規作成時に添付ファイルを追加可能
  - KPI 5 個（総数 / 完了 / 進行中 / 未着手 / 期限超過）を表示
  - 濃紺グラデーションサイドバー + 白基調の SaaS 風 UI に刷新
- カレンダー機能
  - 左サイドメニューに「カレンダー」タブを追加
  - タスクの期限・案件の契約開始月をカレンダー形式で表示
  - 祝日・営業日に対応
  - カレンダーから案件一覧への遷移・ハイライト
- 添付ファイル機能
  - タスク・案件にファイルを添付可能（IndexedDB に Blob としてローカル保存）
  - タスク新規作成時にも添付ファイルを追加可能
  - タスクカード・案件詳細・マインドマップノードに添付アイコンを表示
  - 案件一覧に独立した「添付」カラムを追加
  - PDF はアプリ内プレビュー、それ以外は Finder で開く
- ログイン機能
  - 初回起動時に ID / パスワードを登録
  - セッション管理（日付変更で自動ログアウト）
  - 設定画面から ID / パスワードの変更が可能
- 横断検索
  - 画面上部の検索窓（⌘K / Ctrl+K）でタスク・顧客案件・商流案件・顧客名・業者名を横断検索
  - 検索結果から「案件一覧」「マップ」の2ルートを選択可能
  - 「案件一覧」選択時: 該当行をハイライト表示（詳細モーダルは開かない）
  - 「マップ」選択時: 該当ノードを5秒間パルス発光 → 選択枠に切り替え → 中央配置
  - 顧客案件（pipeline）にも「マップ」ボタンを表示
- 通知機能
  - ヘッダーに通知ベル（件数バッジ付き）を配置
  - 期限超過案件などの通知を一覧表示
  - 通知クリックで該当案件へ遷移
- メモ欄・タスク説明のURLをクリック可能なハイパーリンク化

### 改善

- アプリ全体のヘッダー・サイドバーUIを刷新
  - 検索窓（⌘K）・通知ベル・ユーザーチップ・商流案件入力ボタンを統合配置
- マインドマップの接続線を太く濃く表示（視認性向上）
- 案件一覧のレコードクリックでハイライト表示を追加
- 重複アラート確認モーダル（DuplicateImportModal）を追加

### バグ修正

- 案件一覧で業者（2次卸）削除が保存されない不具合を修正
- 案件一覧のカラム幅はみ出し対応（折り返し表示）

### 内部改善

- Electron 内部を HTTP サーバー方式に移行（app:// → http://localhost:19847）
- IndexedDB の旧originデータを新originへ自動移行
- macOS / Apple Silicon 向けDMGを ad-hoc署名 付きで再生成

## 0.3.3

- 案件一覧テーブルに **カラムリサイズ** を追加
  - 全 16 カラムをユーザーがドラッグで横幅調整可能
  - 顧客名は漢字10文字分（最低 200px）、カテゴリは最低 120px を保証
  - 顧客名が列幅を超える場合は折り返し表示
  - カラム幅は localStorage（キー: `sfa-mvp:dealTable:columnWidths`）に永続化され、次回起動時に復元
- 案件一覧の **メモセル** に **HoverCard** を追加
  - マウスホバー 200ms 後に小さなポップオーバーでメモ全文を表示
  - 最大サイズ 360 × 240 px のスクロール領域
  - スクロール / Esc / マウスがトリガーとポップオーバー両方から外れた時に閉じる
  - 既存デザイン言語（`card` + `shadow-xl`）と統一
- 案件一覧の **テキスト配置** を中央揃えに統一
  - 最上部の 4 セクション（顧客案件 / 商流案件 / 注力カラー設定済 / 期間超過）
  - カラム名
  - レコードの中身
- リサイズハンドルを **デフォルト可視化**
  - 各カラム境界に 1px の薄い縦線（slate-300）
  - hover で accent カラー、drag で accent-strong に強調
  - ヒットエリアは 8px でクリックしやすい
- macOS / Apple Silicon 向けDMGを ad-hoc署名 付きで再生成
