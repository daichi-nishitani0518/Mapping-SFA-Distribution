# Mapping-SFA

Mapping-SFA は、営業パイプラインを整理するための macOS 向けデスクトップアプリです。

## 主な機能

- ダッシュボード
  - 今期パイプライン見込
  - 目標達成率
  - ステータス内訳
  - 月別の積み上げ確認
- マインドマップ
  - 担当者 → 顧客 → 案件 → 業者の関係を可視化
- 案件一覧 / 詳細管理
  - 検索、編集、ステータス確認、案件詳細管理
- バックアップ / 復元
  - ローカル保存データの書き出しと取り込み
- AI用エクスポート
  - AIで扱いやすい形式で案件データを出力
- 差分マージ取り込み
  - 既存データを保ちながら差分取り込み
- 重複候補アラート / 束ね登録
  - 類似案件を検知し、同一案件として束ねるか別案件として扱うかを選択可能

## 対応環境

- macOS
- Apple Silicon 搭載 Mac
