# Mapping-SFA

Mapping-SFA は、営業パイプラインを整理するための macOS 向けデスクトップアプリです。

## 主な機能

- ダッシュボード
  - 今期パイプライン見込
  - 目標達成率（受注確定金額 + 今期パイプライン見込ベース）
  - ステータス内訳（パイプライン中 / 受注確定 / 失注）
  - 月別の積み上げ確認
  - 期間超過案件の最上部アラート表示
- マインドマップ
  - 担当者 → 顧客 → 案件 → 業者の関係を可視化
- 案件一覧 / 詳細管理
  - 検索、編集、ステータス確認、案件詳細管理
  - 詳細画面に今年度計上金額を上部表示
  - 失注変更時の確認ダイアログ（「本当に失注に変更しますか？」）
- バックアップ / 復元
  - ローカル保存データの書き出しと取り込み
- AI用エクスポート
  - AIで扱いやすい形式で案件データを出力
- 差分マージ取り込み
  - 既存データを保ちながら差分取り込み
- 重複候補アラート / 束ね登録
  - 類似案件を検知し、同一案件として束ねるか別案件として扱うかを選択可能
  - 数量±10% / 契約開始年月±3ヶ月以内の束ね条件に対応

## 対応環境

- macOS
- Apple Silicon 搭載 Mac
