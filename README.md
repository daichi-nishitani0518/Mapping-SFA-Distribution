# Mapping-SFA

Mapping-SFA は、営業パイプラインを整理するための macOS 向けデスクトップアプリです。

## 主な機能

- ダッシュボード
  - 今期パイプライン見込
  - 目標達成率（受注確定金額 + 今期パイプライン見込ベース）
  - ステータス内訳（パイプライン中 / 受注確定 / 失注）
  - 月別の積み上げ確認
  - 期間超過案件の最上部アラート表示
- マインドマップ
  - 担当者 → 顧客 → 案件 → 業者の関係を可視化
  - 接続線を太く濃く表示
  - ノードに添付ファイルアイコンを表示
- 案件一覧 / 詳細管理
  - 検索、編集、ステータス確認、案件詳細管理
  - 詳細画面に今年度計上金額を上部表示
  - 失注変更時の確認ダイアログ（「本当に失注に変更しますか？」）
  - カラム幅のドラッグリサイズ（顧客名 200px / カテゴリ 120px を最低保証）
  - メモセルのホバーで全文を HoverCard 表示
  - テキスト配置は中央揃えで一覧性重視
  - レコードクリックでハイライト表示
  - 添付ファイル列（独立カラム）
- タスク管理
  - カンバン形式のタスクボード（カラムカスタマイズ可能）
  - 顧客案件紐付け・優先度・進捗率・期限管理
  - タスク新規作成時に添付ファイルを追加可能
  - KPI 5 個（総数 / 完了 / 進行中 / 未着手 / 期限超過）
  - 濃紺サイドバー + 白基調の SaaS 風 UI
- カレンダー
  - タスク期限・案件契約開始月をカレンダー表示
  - 祝日・営業日対応
  - カレンダーから案件一覧への遷移・ハイライト
- 添付ファイル
  - タスク・案件にファイルを添付可能（IndexedDB にローカル保存）
  - タスク新規作成時にも添付可能
  - PDF はアプリ内プレビュー、それ以外は Finder で開く
- ログイン機能
  - ID / パスワード認証（初回登録制）
  - 日付変更で自動ログアウト
  - 設定画面から ID / パスワード変更可能
- 横断検索
  - 画面上部の検索窓（⌘K / Ctrl+K）でタスク・案件・顧客名・業者名を横断検索
  - 「案件一覧」「マップ」の2ルートから選択可能
  - マップ選択時は該当ノードを発光・中央配置
- 通知機能
  - 期限超過案件などの通知をヘッダーのベルで一覧表示
- バックアップ / 復元
  - ローカル保存データの書き出しと取り込み
- AI用エクスポート
  - AIで扱いやすい形式で案件データを出力
- 差分マージ取り込み
  - 既存データを保ちながら差分取り込み
- 重複候補アラート / 束ね登録
  - 類似案件を検知し、同一案件として束ねるか別案件として扱うかを選択可能
  - 数量±10% / 契約開始年月±3ヶ月以内の束ね条件に対応
- メモ欄・タスク説明のURLリンク化
  - URLがクリック可能なハイパーリンクとして表示

## 対応環境

- macOS
- Apple Silicon 搭載 Mac
